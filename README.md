## Instructions to Run the Project

1. **Clone the repository:**
   ```sh
   git clone https://github.com/rahulsomkunwar73/BigCompany
   cd BigCompany

2. **Build the project using Maven:**
   ```sh
   mvn clean install

3. **Run the program:**
   ```sh
   java -jar target/BigCompany-1.0-SNAPSHOT.jar
